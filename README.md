
## Overview
This project demonstrates lost data retrieval using Recuva and Python automation in a safe Windows NTFS environment.

The experiment simulates accidental file deletion and recovery using digital forensics techniques.

---

# Objectives

- Understand lost data retrieval concepts
- Recover permanently deleted files
- Explore NTFS and FAT32 file systems
- Use Recuva for file recovery
- Verify recovered files using Python

---

# Tools Used

- Recuva
- Python 3
- Visual Studio Code
- Windows 10/11

---

# Project Workflow

1. Create test files
2. Permanently delete files
3. Scan using Recuva
4. Recover deleted files
5. Verify recovery using Python script

---

# Python Automation

The Python script:
- Scans recovered files
- Displays file names
- Displays file sizes
- Generates recovery summary

---

# File System Used

## NTFS
The experiment was performed using the NTFS file system in Windows.
